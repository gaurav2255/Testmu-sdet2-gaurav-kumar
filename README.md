# TestMu SDET2 Automation Framework - Gaurav Kumar

## 📌 Objective
This repository contains a scalable end-to-end test automation framework built using Playwright with JavaScript. The framework is designed to demonstrate strong software testing principles, clean architecture, and maintainable automation practices.

---

## 🧱 Tech Stack
- Playwright (UI + API Testing)
- JavaScript (Node.js)
- Playwright Test Runner
- GitHub Actions (CI/CD)
